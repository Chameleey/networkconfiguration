## 📌 Overview
This project demonstrates the complete setup, configuration, and administration of a Linux server in a virtualized environment. It focuses on deploying core network services, implementing security measures, automating tasks, and monitoring system performance.

---

## ⚙️ Environment Setup
- Ubuntu / Kali Linux  
- Oracle VirtualBox  

---

## 🌐 Network Services Deployment
- DHCP configuration for dynamic IP allocation  
- DNS setup with forward and reverse lookup zones  
- NTP configuration for time synchronization  

---

## 🔐 Security Hardening
- SSH configuration for secure remote access  
- iptables firewall rules for:
  - Web server protection  
  - Application filtering  
  - ICMP control  
  - Controlled printer access  

---

## 🧾 Automation (Bash Scripting)
- Daily system monitoring script (CPU, memory, disk usage)  
- Automated backup script using cron  

---

## 🖥️ Service Deployment
- Apache web server with custom web pages  
- Postfix mail server with Gmail relay  

---

## 📊 Monitoring
- Nagios Core setup for network and service monitoring  

---

## 🛠️ Tools & Technologies
Ubuntu · Kali Linux · VirtualBox · BIND9 · ISC DHCP · iptables · Apache2 · Postfix · Nagios Core · Cron · Bash

Successfully deployed a secure, fully functional Linux server environment capable of delivering multiple network services with automation and monitoring—demonstrating end-to-end system administration and cybersecurity skills.
